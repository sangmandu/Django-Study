# 섹션 9. 탬플릿 파일 모듈화 하기
## extends - 화면 메인 영역 모듈화 하기
한 웹 사이트에서 다른 기능을 제공하는 페이지의 일관성과 더 쉬운 관리를 위해서 모듈화를 수행한다. 모듈화 함수 중에서 extends는 기본이 되는 html파일을
각 페이지에 쓸 수 있게 불러와주는 기능이다.  
예시 :  
```HTML
{% extends 'blog/base.html' %}

{% block main_area %}

{% endblock %}
```
위처럼 {% extends __ %} '__' 칸에 불러올 html 파일경로를 써준다.
